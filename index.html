Snippets Feature Guide

CSS Syntax that you might need every day!

z-index

The z-index property specifies the stack order of an element.

            img{
              position: absolute;
              left: 0px;
              z-index: 2;
            }

hover

Select and style a link when you mouse over it.

            a:hover {
              background-color: yellow;
            }

after

A pseudo-element is created as the last child of the selected element.

            .exciting-text::after {
              content: " <- EXCITING!";
              color: green;
            }

media quaries

The @media defines different style for different media types.

            @media screen and (min-width: 480px) {
              body {
                background-color: lightgreen;
              }
            }

box-shadow

Add shadows to different elements.

            #example1 {
              box-shadow: 5px 10px;
            }

position

It specifies the type of positioning method used for an element.

            div.static {
              position: static;
              border: 3px solid #73AD21;
            }
